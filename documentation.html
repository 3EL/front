CSS

Cascading Style Sheets are used to render and style the User Interface.

<link type="text/css" href="http://3el.github.io/front/dist/front.css" rel="stylesheet">

JavaScript

FAML (Front Attribute Markup Language) and FTML (Front Template Markup Language) uses JavaScript to run.

<script src="http://3el.github.io/front/dist/front.js"></script>
Front attributes
Attribute Description
lib Loads required libraries, see librararies

Starter layout template

The starter layout template is using the holy grail layout as the base and predefined with all related HTML5 elements.

<!doctype html>
<html lang="en">
 <head>
   <title>My application</title>
   <meta http-equiv="X-UA-Compatible" content="ie=edge;ie=10;chrome=1">
   <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
   <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
   <link type="text/css" href="http://3el.github.io/front/dist/front.css" rel="stylesheet">
   <link type="text/css" href="http://3el.github.io/front/dist/design/front/theme/front.css" rel="stylesheet">
   <link type="text/css" href="assets/css/theme.css" rel="stylesheet">
   <script src="http://3el.github.io/front/dist/front.js" lib="xhr;popup;nav;json"></script>
 </head>
 <body>
   <header></header>
   <nav></nav>
   <aside></aside>
   <main include="home.html"></main>
   <aside></aside>
   <footer></footer>
 </body>
</html>
