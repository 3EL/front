Template

Templates are reusable documents that can be included in pages.

Load

Templates will be loaded when template attribute is defined with a valid path.

<script src="../../dist/front.js" title="Template" template="index;template2"></script>
⚠If no template name or true is set, "index" will be used as standard.

Run

The template element will ovveride the starter layout template.

<template>
  <header></header>
  <nav></nav>
  <aside index="0">
  <b>Override content</b>
  </aside>
  <main></main>
  <aside index="1"></aside>
  <footer></footer>
</template>
